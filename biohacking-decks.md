## Импорт

### Anki

1. Сохранить CSV-блок из колоды №2 как `bdnf_cortisol_dopamine_v2.csv` в UTF-8.
2. Anki → *File → Import* → выбрать файл.
3. В диалоге: Field separator = **Semicolon**, Allow HTML — выкл., Field mapping — Front (1), Back (2), Tags (3).
4. Назначить deck «Биохимия: кортизол / BDNF / дофамин v2».
5. Тип карточек — `Basic` (или `Basic (and reversed card)` для обратных).

Колоду №1 (стартовую таблицу) можно вручную перенести в тот же или отдельный deck.

### Obsidian (Spaced Repetition plugin)

Формат `Q::A` — один на строку. Пример:

```
Что такое StAR-белок и зачем он нужен?::Steroidogenic Acute Regulatory protein. Переносит холестерин из внешней мембраны митохондрии во внутреннюю. Это лимитирующий шаг стероидогенеза, не сам фермент.
```

Конвертация CSV → Q::A одной командой в bash:

```bash
awk -F';' 'NR>1 {print $1"::"$2}' bdnf_cortisol_dopamine_v2.csv > biohacking.md
```

Или в Python:

```python
import csv
with open("bdnf_cortisol_dopamine_v2.csv", encoding="utf-8") as fin, \
     open("biohacking.md", "w", encoding="utf-8") as fout:
    next(fin)
    for row in csv.reader(fin, delimiter=";"):
        fout.write(f"{row[0]}::{row[1]}\n")
```

## Заметки для следующего чата

- Колоды лучше прокачивать последовательно: 2 недели на №1, потом подключать №2. Не зашивать одновременно.
