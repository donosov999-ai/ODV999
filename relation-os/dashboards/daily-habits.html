    Что такое B = MAP и зачем 60-90 дней

    B = MAP (Fogg): Поведение происходит = Motivation × Ability × Prompt. Если поведения нет — слабо одно из трёх. Микропривычка решает все три: motivation = маленький порог, ability = занимает 60 секунд, prompt = триггер (после X делаю Y).

    Hebbian rule: «Нейроны, активирующиеся вместе, связываются». 60-90 дней повторений = автоматизм.

    Embodied cognition: Тело меняет мозг быстрее, чем разговор. Ежедневное «6-сек поцелуй» переписывает связь «утро → телефон» на «утро → телесная близость с партнёром» — за месяц.
