Что это

Это шаблон дашборда. Если ты видишь эту страницу по адресу https://donosov999-ai.github.io/ODV999/dashboards/example.html — значит GitHub Pages включён и всё работает.

Что сюда можно добавить

  • 📊 Графики — Chart.js / D3 / ECharts (через CDN)
  • 📋 Таблицы — DataTables или нативный HTML с поиском
  • 🔍 Фильтры — обычный JS (не нужен бэкенд)
  • 📦 Данные — статичный JSON в data/
  • 🌐 Внешние API — fetch к публичным эндпоинтам

Как сделать новый дашборд

  1. Скопируй этот файл: cp dashboards/example.html dashboards/my-new.html
  2. Поправь содержимое
  3. Добавь карточку в index.html (раздел <!-- DASHBOARDS LIST -->)
  4. git add . && git commit -m "Add my-new dashboard" && git push
  5. Жди 1-2 минуты — обновится автоматически
